# Contributing to Nurovia

Thanks for your interest in contributing! 🎉
We're a small team, so every PR, issue, or doc fix genuinely helps.

## 🧭 Ground rules

- Be respectful. Read our [Code of Conduct](./CODE_OF_CONDUCT.md).
- Search existing issues before opening a new one.
- Keep PRs focused — one fix or feature per PR.

## 🛠️ How to contribute

1. **Fork** the repo
2. Create a branch: `git checkout -b fix/short-description`
3. Make your changes
4. Test locally
5. **Open a PR** using the [PR template](./.github/PULL_REQUEST_TEMPLATE.md)
6. Respond to review feedback within a reasonable window

## 📦 Commit conventions

We use [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add anime search to Mika
fix(medicore): resolve patient record sync bug
docs: update README with new install steps
chore: bump dependencies
```
